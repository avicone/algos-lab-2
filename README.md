# algos-lab-2


* Описание проекта:
1) Проект реализует систему автобусного маршрута с использованием линейной структуры данных — односвязного списка.

2) Каждая остановка хранит:
    1) название остановки;
    2) координаты;
    3) время до следующей остановки;
-----------------------------------------------------------------------------------------------------------------------------------
* Реализованные функции
Обязательная часть:
1) Добавление остановки

    Пользователь может добавлять новые остановки в маршрут.

2) Расчет общего времени маршрута

    Программа вычисляет сумму времени между всеми остановками.

3) Определение местоположения автобуса через N остановок

    Пользователь выбирает текущую остановку и количество остановок, после чего программа определяет, где окажется автобус.

4) Построение обратного маршрута

    Маршрут может быть развернут в обратном направлении.

-----------------------------------------------------------------------------------------------------------------------------------
* Вариативная часть  
3.**Добавить расчет времени прибытия (расширение функции "где будет автобус через N остановок")**  
  Функция работает по принципу маятника (челночное движение):
  Изначально мы выбираем остановку, от которой будем считать куда он попадет потом. В случае если он дошел до последней, он разворачивается и идет в обратном направлении, доходя до первой снова разворачивается. Т.е. у нас есть остановки 1,2,3. Если взять как начальную точку остановку 2 и отсчитать от нее 2 остановки, то автобус вернется на нее.  
5.**Вывести маршрут в виде таблицы с выравниванием колонок**  
  Маршрут выводится в виде таблицы, именно под ней выводится общее время маршрута. Также выводятся координаты, время до следующей остановки и время пройденное от начала маршрута до станции.
